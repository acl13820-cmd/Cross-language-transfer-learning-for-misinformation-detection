# QMSS Thesis: Cross-language Transfer Learning for Chinese, Japanese, and English Misinformation Detection
This repository stores the notebooks that documented the following steps:
 * Data processing
 * NLP emotion detection
 * Machine learning

## Data sources: 
* Chinese language misinformation data:
  * [MCFEND: A Multi-source Benchmark Dataset for Chinese Fake News Detection]([url](https://arxiv.org/abs/2403.09092))
* English language misinformation data:
  * [WELFake dataset for fake news detection in text data]([url](https://zenodo.org/records/4561253))
* Japanese language misinformation data: (created with two separate datasets)
  * [Japanese Fake News Dataset | Taichi Murayama]([url](https://hkefka385.github.io/dataset/fakenews-japanese/))
  * [GitHub - tanreinama/Japanese-Fakenews-Dataset: 日本語フェイクニュースデータセット]([url](https://github.com/tanreinama/Japanese-Fakenews-Dataset))
